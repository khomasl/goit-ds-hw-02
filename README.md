# Tasks System Management

=======================

## How to Use

Clone

```bash
git clone https://github.com/khomasl/goit-ds-hw-02.git
```

Then

```bash
cd goit-ds-hw-02
```

## Install and Activate virtual environment

```bash
poetry install
poetry shell
```

## Create database and tables

```bash
python create_db.py
```

## Fill tables with fake data

```bash
python fill_tables.py
```

## Do some queries

```bash
python queries.py
```
